# RickBroider.com

Website foundation for Rick Broider's personal brand ecosystem: regenerative systems, conscious AI, operational coherence, diagnostics, frameworks, and related projects.

## Current Setup

This is currently a dependency-free static site. Open `index.html` directly in a browser, or host the repository root with any static host.

## Current Scope

- Dependency-free static site baseline
- Semantic landing page sections based on the website brief
- Person schema and social metadata in `index.html`
- Responsive visual system for a grounded future-human brand direction
